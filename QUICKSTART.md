# 🚀 Быстрый запуск Delegator

## Шаг 1: Установка PostgreSQL

### Windows
1. Скачайте PostgreSQL с https://www.postgresql.org/download/windows/
2. Установите, запомните пароль для пользователя postgres
3. Откройте pgAdmin и создайте базу данных:
   ```sql
   CREATE DATABASE delegator;
   ```

## Шаг 2: Настройка проекта

### 2.1 Установка зависимостей
```bash
cd I:\site\delegator\delegator
npm install
```

### 2.2 Настройка .env файла

Откройте файл `.env` и измените:

```env
DATABASE_URL="postgresql://postgres:ВАШ_ПАРОЛЬ@localhost:5432/delegator?schema=public"
JWT_SECRET="super-secret-key-change-this-in-production-12345"
```

## Шаг 3: Применение схемы базы данных

```bash
npx prisma migrate dev --name init
```

Или для быстрого создания таблиц без миграций:
```bash
npx prisma db push
```

## Шаг 4: Запуск проекта

### Режим разработки (с авто-обновлением)
```bash
npm run dev
```

Откройте http://localhost:3000

### Продакшен режим
```bash
npm run build
npm start
```

## 📝 Регистрация первого пользователя

1. Откройте http://localhost:3000
2. Нажмите "Регистрация"
3. Заполните данные:
   - Имя: Иван
   - Фамилия: Иванов
   - Email: director@company.com
   - Пароль: password123
   - Роль: Директор

4. После регистрации вы попадете в панель директора

## ➕ Добавление сотрудника

1. В панели директора перейдите в "Сотрудники"
2. Нажмите "Добавить сотрудника"
3. Заполните данные сотрудника
4. Сотрудник сможет войти под своими данными

## 📋 Создание задачи

1. В панели директора перейдите в "Задачи"
2. Нажмите "Создать задачу"
3. Заполните:
   - Название
   - Описание
   - Приоритет (Низкий, Средний, Высокий, Срочно)
   - Срок исполнения (опционально)
   - Исполнителя (опционально)

## 📊 Структура базы данных

### Таблицы:
- **User** - Пользователи (директора и сотрудники)
- **Company** - Компании
- **Task** - Задачи
- **Comment** - Комментарии к задачам

### Роли:
- **DIRECTOR** - Может создавать задачи и сотрудников
- **EMPLOYEE** - Может только просматривать назначенные задачи

### Статусы задач:
- **PENDING** - Ожидает
- **IN_PROGRESS** - В работе
- **COMPLETED** - Завершено
- **CANCELLED** - Отменено

### Приоритеты:
- **LOW** - Низкий
- **MEDIUM** - Средний
- **HIGH** - Высокий
- **URGENT** - Срочно

## ❓ Решение проблем

### Ошибка подключения к БД
- Проверьте, что PostgreSQL запущен
- Проверьте пароль в DATABASE_URL
- Убедитесь, что база данных delegator существует

### Ошибка Prisma
```bash
npx prisma generate
npx prisma db push
```

### Ошибка портов
Если порт 3000 занят, запустите на другом:
```bash
npm run dev -- -p 3001
```
